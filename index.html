📚 第01課 重要文型
文型1　제 이름은 -예요 / -이에요
-이다は名詞と結合して、主語と述語が同じであることを意味する。
直前の名詞の語末にパッチムあり → -이에요
直前の名詞の語末にパッチムなし → -예요
제 이름은 김영희예요. （영희 → 받침 없음）
→ 私の名前はキム・ヨンヒです。
저는 학생이에요. （학생 → 받침 ㅇ）
→ 私は学生です。
文型2　저는 / 은·는 (주제 조사)
저は1人称の謙譲語。日本語の「私（わたくし）」に相当する。
-은/는は日本語の「-は」のような主題を表す副助詞。
パッチムあり → -은　パッチムなし → -는
저는 회사원이에요. （저 → 받침 없음 → 는）
→ 私は会社員です。
이름은 뭐예요? （이름 → 받침 ㅁ → 은）
→ お名前は何ですか？
文型3　-사람이에요? / -사람이 아니에요
国籍を聞く・答える表現。否定は-이/가 아니에요を使う。
가: 한국 사람이에요?
→ 韓国人ですか？
나: 아니요. 일본 사람이 아니에요.
→ いいえ。日本人ではありません。
文型4　-도 （〜も）
-도は「〜も」を意味する。同じ質問をする場面で、主格助詞-이/가や主題を表す副助詞-은/는の代わりに使われる。
히로 씨도 학생이에요?
→ 廣さんも学生ですか？
네. 저도 학생이에요.
→ はい。私も学生です。

一人で重荷を背負って生きる中年男性と、生きることに疲れた若い女性が互いを癒していく深い人間ドラマ。日常の韓国語とあいさつ表現が自然に学べます。
📚 학습 포인트
안녕하세요（こんにちは）、감사합니다（ありがとう）など今回学んだあいさつが頻繁に登場します。職業や自己紹介の表現も確認できます。
🎵 노래: IU - 좋은 날 (Good Day)
韓国を代表する国民的シンガーIUの代表曲。明るくテンポよい歌詞で、日常の韓国語表現が自然に耳に入ってきます。
📚 학습 포인트
「나는」（私は）「좋아」（好き・いい）など-은/는 조사와 기본 형용사 표현을 노래로 익힐 수 있습니다。
