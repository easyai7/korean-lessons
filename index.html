📚 重要文型
文型1　N + 은/는（主題の助詞）
日本語の「～は」に相当する주題の助詞。
パッチムあり → -은　パッチムなし → -는
저는 나오키예요. (私は直樹です。)
나오키 씨는요? (直樹さんは？)
文型2　N + 이에요/예요（～です）
名詞の後についてです・ますの意味を表す丁寧形。
パッチムあり → -이에요　パッチムなし → -예요
회사원이에요. ← 회사원にパッチムあり
会社員です。
대학생이에요. ← 대학생にパッチムあり
大学生です。
수정이에요. / 나오키예요.
スジョンです。/ 直樹です。
文型3　뭐예요? / N이/가 뭐예요?（何ですか？）
뭐예요？は「何ですか？」という疑問문。
名詞 + 이/가 뭐예요？の形でよく使われる。
직업이 뭐예요? (お仕事は何ですか？)
이름이 뭐예요? (お名前は何ですか？)
文型4　어느 나라 사람이에요?（どこの国の人ですか？）
어느はどの・どちらの意味。국적を尋ねる時は어느 나라 사람이에요？を使う。
어느 나라 사람이에요? → 한국 사람이에요.
どこの国の人ですか？→ 韓国人です。
일본 사람이에요. / 미국 사람이에요.
日本人です。/ アメリカ人です。
⭐ 韓流情報
🇰🇷 한국의 인사 문화 (韓国の挨拶文化)
韓国では初対面の挨拶で처음 뵙겠습니다と言いながら軽くお辞儀をします。また、名刺を두 손으로 주고받는 것、両手で渡し受け取ることがマナーです。目上の人には반말、タメ말ではなく존댓말、敬語を必ず使います。
📚 인사 표현
안녕하세요.（こんにちは。）
안녕히 가세요.（さようなら。〈残る人が言う〉）
안녕히 계세요.（さようなら。〈去る人が言う〉）
🎬 드라마: 이상한 변호사 우영우 (ウ・ヨンウ弁護士は天才肌)
天才的な記憶력을 가진 자폐스펙트럼 변호사의 성장 드라마。日本でも大人気のこの드라마では、主人公が毎回저는 우영우입니다、私はウ・ヨンウですと自己紹介するシーンが印象的です。

✝️ 선교 메모 (宣教メモ)
일본에서 한국어를 가르치는 것은 단순한 언어 교육이 아닙니다。저는 선교사입니다라고 自己紹介できる自然な機会が生まれます。처음 뵙겠습니다という一言から始まる関係が、福音를 나누는 기회로 이어질 수 있습니다。잘 부탁드립니다는 하나님의 은혜를 구하는 기도와 같은 마음으로도 사용할 수 있습니다。
