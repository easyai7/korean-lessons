💡 連音（リョンウム）とは？
パッチムのある文字の後に母音が続く場合、パッチムの子音は次の音節と一緒に発音される。
例）이름이 → [이르미]　사람이에요 → [사라미에요]

📚 第01課 重要文型
文型1　제 이름은 -예요 / -이에요
-이다は名詞と結合して、主語と述語が同じであることを意味する。
直前の名詞の語末にパッチムあり → -이에요
直前の名詞の語末にパッチムなし → -예요
제 이름은 김영희예요. （영희 → 받침 없음）
→ 私の名前はキム・ヨンヒです。
저는 학생이에요. （학생 → 받침 ㅇ）
→ 私は学生です。
文型2　저는 / 은·는 (주제 조사)
저は1人称の謙譲語。日本語の「私（わたくし）」に相当する。
-은/는は日本語の「-は」のような主題を表す副助詞。
パッチムあり → -은　パッチムなし → -는
저는 회사원이에요. （저 → 받침 없음 → 는）
→ 私は会社員です。
이름은 뭐예요? （이름 → 받침 ㅁ → 은）
→ お名前は何ですか？
文型3　-사람이에요? / -사람이 아니에요
国籍を聞く・答える表現。否定は-이/가 아니에요を使う。
가: 한국 사람이에요?
→ 韓国人ですか？
나: 아니요. 일본 사람이 아니에요.
→ いいえ。日本人ではありません。
文型4　-도 （〜も）
-도は「〜も」を意味する。同じ質問をする場面で、主格助詞-이/가や主題を表す副助詞-은/는の代わりに使われる。
히로 씨도 학생이에요?
→ 廣さんも学生ですか？
네. 저도 학생이에요.
→ はい。私も学生です。

🎵 노래: IU - 좋은 날 (Good Day)
韓国を代表する国民的シンガーIUの代表曲。明るくテンポよい歌詞で、日常の韓国語表現が自然に耳に入ってきます。
📚 학습 포인트
