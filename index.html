📅 第09課 — 2호선에서 3호선으로 갈아타야 해요.
📱 新出単語
💡 発音変化のポイント（第09課）
激音化：ㄱ・ㄷ・ㅂ・ㅈ の前後に ㅎ がある場合 → ㅋ・ㅌ・ㅍ・ㅊ
어떻게 → [어떠케]　좋다 → [조타]
濃音化：직접 → [직쩝]
ㄴ添加：안국역 → [안궁녁]
지하철 / 버스
뜻: 地下鉄 / バス
読み: チハチョル / ポス
💡 지하철로 가요（地下鉄で行きます）— 「로」は手段を表す！
갈아타다
뜻: 乗り換える
読み: カラタダ
💡 갈아타야 해요 → [가라타야 해요]　乗り換えなければなりません
내리다
뜻: 降りる
読み: ネリダ
💡 어디에서 내려요？（どこで降りますか？）
출구
뜻: 出口
読み: チュルグ
💡 3번 출구로 나가면 돼요（3番出口を出ればいいです）
어떻게
뜻: どうやって・どのように
読み: [어떠케] ← 激音化に注意！
💡 표기는 어떻게이지만 발음은 [어떠케]！
직접
뜻: 直接
読み: [직쩝] ← 濃音化！
💡 직접 가는 버스가 없어요（直接行くバスがありません）
얼마나 걸려요？
뜻: どれくらいかかりますか？
読み: オルマナ コルリョヨ
💡 얼마나=どれくらい　(시간이) 걸리다=（時間が）かかる
20분쯤
뜻: 約20分
読み: イシップ・プンチュム
💡 쯤=〜くらい・約。지하철로 20분쯤 걸려요（地下鉄で約20分かかります）
📝 会話
여기서 인사동은 어떻게 가요?
뜻: ここから仁寺洞はどうやって行きますか？
💡 어떻게 → [어떠케]（激音化）。여기서=ここから
직접 가는 버스가 없어요. 저쪽에서 지하철을 타야 해요.
뜻: 直接行くバスはありません。あちらで地下鉄に乗らなければなりません。
💡 직접 → [직쩝]（濃音化）　타야 해요＝乗らなければならない
어디에서 내려요?
뜻: どこで降りますか？
💡 어디=どこ　에서=で（動作の場所）　내려요=降ります
3호선 안국역에서 내려서 6번 출구로 나가면 돼요.
뜻: 3号線安国駅で降りて6番出口を出ればいいです。
💡 안국역 → [안궁녁]（ㄴ添加）　내려서=降りて　나가면 돼요=出ればいい
여기서 인사동까지 얼마나 걸려요?
뜻: ここから仁寺洞までどれくらいかかりますか？
💡 까지=〜まで（終点）　얼마나=どれくらい　걸려요=かかります
지하철로 20분쯤 걸려요.
뜻: 地下鉄で約20分かかります。
💡 지하철로=地下鉄で（手段）　쯤=〜くらい・約　걸려요=かかります
📚 第09課 重要文型
文型33　-아야/어야/여야 해요（〜しなければならない）
用言の語幹に -(아/어/여)야 해요 を付けて「〜しなければならない」という意味を表す。
語幹末の母音がㅏ・ㅗ → -아야 해요　それ以外 → -어야 해요　하다 → -해야 해요
지하철을 타야 해요. （타다 → 가야 해요と同形）
→ 地下鉄に乗らなければなりません。
신분증을 준비해야 해요. （준비하다 → 해야 해요）
→ 身分証を準備しなければなりません。
책을 읽어야 해요. （읽다 → 어야 해요）
→ 本を読まなければなりません。
文型34　-(아/어/여)서（〜して：時間的順序）
動詞語幹に -(아/어/여)서 を付けて時間的な順序「〜して」を表す。
先行節と後続節の主語は一致しなければならない。
안국역에서 내려서 6번 출구로 나가면 돼요.
→ 安国駅で降りて6番出口を出ればいいです。
서점에 가서 책을 샀어요.
→ 本屋に行って本を買いました。
文型35　-(으)면 돼요（〜ばいい）
用言の語幹に -(으)면 돼요 を付けて「〜ばいい・〜ばよい」を表す。
語幹末がパッチムあり → -으면 돼요　パッチムなし・ㄹ → -면 돼요
3번 출구로 나가면 돼요. （나가다 → 파っ침なし → 면）
→ 3番出口を出ればいいです。
이 약은 하루에 세 번 먹으면 돼요. （먹다 → パッチムㄱ → 으면）
→ この薬は一日に3回飲めばいいです。
文型36　-(으)로（〜で：手段・方法）
手段・方法を表す助詞。
直前の名詞が母音・ㄹ終わり → -로　その他の子音終わり → -으로
지하철로 20분쯤 걸려요. （지하철 → ㄹ終わり → 로）
→ 地下鉄で約20分かかります。
버스로 가면 돼요. （버스 → 母音終わり → 로）
→ バスで行けばいいです。
여객선으로 가요. （여객선 → ㄴ終わり → 으로）
→ 旅客船で行きます。
⭐ 韓流情報
🚇 서울 지하철 (ソウル地下鉄)
ソウルの地下鉄は9路線以上あり、世界でも有数の公共交通網を誇ります。観光スポットへのアクセスも抜群で、韓国語の案内放送を聞き取るだけで、今課の표현が自然に身につきます。

「○호선으로 갈아타세요」（○号線に乗り換えてください）
🎬 드라마: 도깨비 (鬼 – Guardian)
孤独な神・「鬼」と人間の運命を描いた大ヒットファンタジードラマ。ソウルの街並みや地下鉄などの移動シーンも多く登場し、方向や交通手段に関する자연스러운 표현が随所に出てきます。
📚 학습 포인트
「어디 가요？」（どこ行くんですか？）
「여기서 멀어요？」（ここから遠いですか？）
移動・道案内表現がたっぷり学べます。
🍜 한국 음식: 인사동 길거리 음식
仁寺洞（インサドン）は伝統工芸品やギャラリーが並ぶ文化の街。今課の会話の目的地でもあります。散策しながら韓国の길거리 음식（屋台グルメ）を楽しめます。
